# Instagram Follower Bot

An automated Python script built with **Selenium** that logs into a specified Instagram account, navigates to a target profile, and automatically follows its followers. Perfect for automating networking and growth target analysis.

---

## 🚀 Features

* **Automated Login:** Securely handles user authentication to log into Instagram.
* **Target Scraper:** Navigates directly to any target Instagram profile provided.
* **Follow Automation:** Opens the target's follower list and handles the scrolling and clicking mechanics to follow accounts seamlessly.
* **Built with Selenium:** Utilizes modern browser automation techniques.

---

## 🛠️ Prerequisites

Before running the script, ensure you have the following installed:

* **Python 3.x**
* **Google Chrome** (and the matching version of ChromeDriver, or let Selenium handle it automatically via webdriver-manager)
